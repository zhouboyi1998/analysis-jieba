### ⌛ 开始

#### 安装第三方库

```
pip3 install requests

pip3 install bs4

pip3 install jieba
```

#### 使用方法

* 运行 `/bin/start.py` 文件
    * `python start.py`
* 获取到的爬取数据和统计数据存放在 `/data` 目录下
